# 每日锻炼 - Daily Exercise

一个基于 Vue3 的每日锻炼跟练 Web 应用，帮助你科学地坚持每日锻炼计划。

## 功能特点

- 📅 **周计划展示**：完整的 7 天锻炼计划，每天不同的训练主题
- ⏱️ **智能计时器**：动作时间和休息时间自动倒计时
- 🔄 **自动切换**：组间和动作间自动切换，无需手动操作
- 📊 **进度追踪**：实时显示训练进度，记录完成情况
- 🏆 **成就系统**：连续完成天数统计，激励坚持
- 📱 **响应式设计**：完美适配移动端和桌面端

## 训练计划

- **周一**：全身燃脂入门
- **周二**：核心 + 腰腹
- **周三**：轻量有氧 + 放松
- **周四**：下肢 + 翘臀
- **周五**：全身综合燃脂
- **周六**：肩背 + 体态
- **周日**：完全休息/轻拉伸

## 技术栈

- Vue 3 - 渐进式 JavaScript 框架
- Vite - 下一代前端构建工具
- Pinia - Vue 状态管理
- Vue Router - 官方路由管理器

## 项目结构

```
daily-exercise/
```

## 开发

### 安装依赖

```bash
pnpm install
```

### 启动开发服务器

```bash
pnpm run dev
```

### 构建生产版本

```bash
pnpm run build
```

### 预览生产构建

```bash
pnpm run preview
```

## 使用方法

1. 打开应用，查看今日训练计划
2. 点击"开始训练"按钮
3. 跟随计时器完成每个动作
4. 完成所有训练后查看成就统计

## 本地存储

应用使用 localStorage 保存以下数据：
- 训练历史记录
- 连续完成天数
- 训练进度
